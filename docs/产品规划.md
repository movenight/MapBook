# MapBook 产品规划文档

## 1. 产品概述

### 1.1 产品定位
MapBook 是一款基于 Web 浏览器的轻量级路书工具，帮助用户在地图上规划旅行路线、管理每日行程，并以直观的方式总览和编辑整个旅程。

### 1.2 目标用户
- 自驾旅行爱好者：规划长途自驾路线
- 旅行攻略作者：创建和分享行程
- 骑行者/徒步者：规划运动路线

### 1.3 核心价值
- 在地图上直观创建和编辑路线
- 按天管理行程，一目了然
- 轻量、快速、无需安装

## 2. 功能模块

### 2.1 地图交互 (P0)
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 地图显示与操作 | 高德地图底图，支持缩放、拖拽、卫星视图切换 | P0 |
| 地点搜索 | 输入关键词搜索 POI，支持模糊匹配 | P0 |
| 点击添加地点 | 直接在地图上长按/右键添加地点标记 | P0 |
| 出发地/目的地标记 | 用不同图标区分出发地和目的地 | P0 |
| 导航路线生成 | 选择两个地点自动生成驾车/骑行/步行路线 | P0 |

### 2.2 行程管理 (P0)
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建路书 | 新建空白路书，输入标题、描述 | P0 |
| 按天规划 | 将路线拆分为多天，每天可添加多个途径点 | P0 |
| 拖拽排序 | 拖拽调整每天内地点顺序 | P0 |
| 跨天移动 | 将某个地点拖到另一天 | P1 |
| 添加途径点 | 在已有路线上插入中间停留点 | P0 |
| 删除地点 | 删除某天的某个地点 | P0 |
| 路线总览 | 在地图上用不同颜色显示每天路线 | P0 |

### 2.3 路书管理 (P1)
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 路书列表 | 卡片式展示所有路书，含封面图、标题、日期 | P1 |
| 路书编辑 | 打开已有路书继续编辑 | P1 |
| 路书删除 | 删除不需要的路书 | P1 |
| 路书复制 | 复制已有路书作为新路书模板 | P2 |
| 导出分享 | 导出为图片或分享链接 | P2 |

### 2.4 用户系统 (P1)
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 邮箱注册/登录 | 支持邮箱密码注册 | P1 |
| 第三方登录 | 微信/GitHub OAuth 登录 | P2 |
| 个人中心 | 查看个人信息、编辑头像昵称 | P2 |

### 2.5 辅助功能 (P2)
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 距离/时长统计 | 自动计算每天行程距离和预计时间 | P1 |
| 天气查看 | 显示目的地天气 | P2 |
| 住宿标记 | 标记每晚住宿地点 | P1 |
| 备注信息 | 每个地点可添加文字备注 | P1 |
| 深色模式 | 支持浅色/深色主题切换 | P2 |

## 3. 版本规划

### V1.0 (MVP) — 核心路书创建
- 地图交互（搜索 + 标记 + 路线生成）
- 创建/编辑路书
- 按天规划行程
- 地点增删排序
- 路线总览展示
- 邮箱注册/登录
- 路书列表与管理

### V1.1 — 体验优化
- 拖拽排序（天与天之间）
- 住宿标记 + 备注信息
- 距离/时长自动统计

### V2.0 — 扩平台
- 响应式适配移动端 Web
- 微信小程序版本（基于 Uni-app）
- 分享功能
- 深色模式

## 4. 非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 首屏加载 < 3s，地图操作帧率 ≥ 30fps |
| 兼容性 | Chrome 90+, Edge 90+, Safari 15+ |
| 安全性 | HTTPS + Supabase RLS 行级安全策略 |
| 可扩展 | 架构预留 Uni-app 移植接口，组件化设计 |
| 轻量 | 打包体积 < 500KB (gzip)，不包括地图 SDK |

---

## 5. 设计约束

1. **地图选型**：高德地图 JS API 2.0，需申请 Web 端 JS API Key
2. **后端选型**：Supabase（开源 BaaS）
   - 数据库：PostgreSQL + RLS 行级安全
   - 认证：Supabase Auth（邮箱 + OAuth）
   - 存储：Supabase Storage（封面图等静态资源）
3. **前端框架**：Vue 3 + Vite + TypeScript
4. **UI 库**：Element Plus（按需引入，保持轻量）
5. **状态管理**：Pinia
6. **路由**：Vue Router 4
